# PayKit

> Multi-tenant invoicing & payment collection platform

Business owners create invoices, share payment links, collect payments via Razorpay, and withdraw earnings to their bank/UPI.

## Repositories

| Repo | Description | Stack |
|------|-------------|-------|
| [Paykit](https://github.com/paykit-io/Paykit) | Backend API | Java 21, Spring Boot 3.2, PostgreSQL |
| [paykit-frontend](https://github.com/paykit-io/paykit-frontend) | Frontend SPA | React 19, TypeScript, Vite |

## Tech Stack
Java 21 · Spring Boot · PostgreSQL · Redis · Razorpay · React · TypeScript
